# Nuvolino | Web App meteo
Web App responsive che integra diverse API per visualizzare condizioni meteo attuali, orarie e giornaliere (fino a 10 giorni), con ricerca città tramite digitazione, geolocalizzazione o navigazione su mappa.  
Tecnologie: HTML, Tailwind CSS, React.js, Custom Hooks, Fetch API, gestione timezone con Luxon
